## Datos crudos

Esta carpeta contiene el conjunto de datos original utilizado en el proyecto.

- Dataset: Superfund National Priorities List (NPL) – Sites with Status Information
- Fuente: Environmental Protection Agency (EPA) – Estados Unidos
- Formato: CSV
- Licencia: Datos públicos

El archivo no fue modificado. Todas las tareas de limpieza, transformación y análisis se realizaron en Google Sheets.
