# Análisis de Remediación de Sitios Superfund

## Descripción del proyecto

Este proyecto analiza datos públicos del programa Superfund de la Environmental Protection Agency (EPA) de Estados Unidos, con el objetivo de evaluar los tiempos y estados de remediación de sitios contaminados con hidrocarburos.

El análisis busca identificar patrones, demoras y posibles cuellos de botella en proyectos de remediación ambiental, utilizando herramientas de análisis de datos en Google Sheets.

## Conjunto de datos

- Fuente: Environmental Protection Agency (EPA)
- Dataset: Superfund National Priorities List (NPL) – Sites with Status Information
- Formato: CSV
- Ubicación: `/datos/crudos`

## Herramientas utilizadas

- Google Sheets
- GitHub
